# Application Goal

## Product mission

Build a lightweight, local-first study companion that helps candidates prepare for the **Databricks Data Engineer Associate** exam through daily execution tracking and targeted quiz practice.

## Primary user

- A learner preparing for the Databricks Data Engineer Associate exam.
- Often studying solo, in short sessions, and needing visible progress + weak-area feedback.

## What the app should optimize for

- **Consistency:** Encourage daily study completion (Day 0–Day 7 weekly execution).
- **Retention:** Reinforce concepts through focused and mixed quiz practice.
- **Feedback loop:** Surface weak topics with clear, official-study-resource next steps.
- **Simplicity:** Fast, no-account workflow with local persistence and immediate resume.

## Success signals

- User can quickly mark and review weekly study execution.
- User can start/continue/finish quiz sessions without friction.
- User receives understandable performance feedback and actionable links.
- App remains stable and easy to use across desktop/mobile in a local-only setup.
