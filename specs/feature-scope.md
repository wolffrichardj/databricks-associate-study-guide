# Feature Scope (v1)

## In scope

- Weekly study tracker with Day 0–Day 7 completion state and persistence.
- Quiz setup and play flow for:
  - Focused topic practice.
  - Mixed/exam-weighted practice.
- Weak-topic analysis and prioritized study resource links.
- Resume active quiz sessions on the same device.
- Program reset action for clearing tracked state.

## Out of scope (current)

- Authentication or user accounts.
- Cloud sync or backend storage.
- Multi-user collaboration.
- Advanced analytics dashboards beyond current progress/performance views.

## Constraints and principles

- Local-first: state is persisted on-device.
- Lightweight UX: minimal steps to start studying.
- Databricks-focused content/resources.
- Changes should preserve clear separation of weekly, quiz, and progress experiences.
